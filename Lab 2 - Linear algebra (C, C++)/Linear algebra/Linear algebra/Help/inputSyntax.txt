	Правила оформления входного файла

Каждое задание обязательно должно начинаться с символа "№".
За символом "№", без пробелов, должно следовать название задания.
Название задания состоит из двух чисел разделенных точкой:
первое число — номер темы,
второе число — номер задачи в данной теме.

После объявления задания должны следовать входные данные,
необходимые для выполнения данной задачи.
(Для ознакомления с правилами оформления отдельных задач нажмите "+").

После входных данных обязательна постановка точки с запятой(;).
Каждое задание должно начинаться с новой строки.

Неправильное оформление входного файла может привести к непредвиденным ошибкам,
а возможно и краху программы.
Неправильное оформления входного файла грозит уголовной ответственностью!
